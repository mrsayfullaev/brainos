# Версионирование BrainOS

Формат версии: **MAJOR.MINOR.PATCH** (например, 4.0.0).

---

## Правила изменения версии

| Изменение | Новая версия | Пример |
|-----------|--------------|--------|
| **Добавление или удаление модулей** | MAJOR + 1 | 4.0.0 → **5.0.0** |
| **Переделывание логики существующих модулей** | MINOR + 1 | 4.0.0 → **4.1.0** |
| **Мелкая правка внутри модуля** | PATCH + 1 | 4.0.0 → **4.0.1** |

---

## Примеры

| Действие | Версия |
|----------|--------|
| Добавили модуль Research | 5.0.0 |
| Удалили модуль News | 5.0.0 |
| Переделали роутер (передача action в handlers) | 4.1.0 |
| Изменили логику парсера wallet | 4.1.0 |
| Исправили баг в task handler | 4.0.1 |
| Обновили текст в инструкции | 4.0.1 |
| Подключили handleShowTasks в task | 4.0.1 (если без смены логики) или 4.1.0 (если меняем архитектуру) |
