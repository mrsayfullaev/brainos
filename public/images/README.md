# Публичные изображения

Файлы в этой папке доступны по URL: `https://ваш-домен/images/имя-файла`.

## Какие картинки нужны и куда загружать

| Назначение | Имя файла | Куда загрузить |
|------------|-----------|----------------|
| **Шапка:** BrainOS с иконкой (одна картинка целиком) | `logo-brainos.png` или `logo-brainos.svg` | Папка **`public/images/`** |
| **Футер, слева:** Душный | `dushny.png` (или `.svg`) | Папка **`public/images/`** |
| **Футер, справа:** Djamshid Aihan | `djamshid-aihan.png` (или `.svg`) | Папка **`public/images/`** |

- В разметке главной страницы указаны: `/images/logo-brainos.png`, `/images/dushny.png`, `/images/djamshid-aihan.png`. Если будете использовать SVG, замените в `public/index.html` расширение на `.svg` в соответствующих тегах `<img>`.
- Рекомендуемая высота в макете: лого в шапке ~40px, картинки в футере ~48px (ширина — по пропорциям).

## Другие форматы

Поддерживаются: SVG, PNG, JPG, JPEG, WebP, ICO.
